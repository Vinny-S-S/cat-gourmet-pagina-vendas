# Cat Gourmet - Landing Page

Este projeto é uma **landing page** desenvolvida para apresentar a ração premium **Cat Gourmet**, destacando seus diferenciais, benefícios e captando leads de clientes interessados.

## 🎯 Objetivo
Criar uma página atrativa e responsiva que:
- Mostre os ingredientes naturais e frescos da ração.
- Apresente os benefícios comprovados para os gatos.
- Incentive o usuário a entrar em contato ou solicitar mais informações.
- Transmita sofisticação e confiança na marca.

## 🖼️ Estrutura da Página
- **Cabeçalho fixo** com logo e navegação para as seções principais.
- **Hero Section** com imagem de destaque e chamada para ação.
- **Seção de Ingredientes** em formato de cards alternados (imagem à direita/esquerda), com bordas arredondadas e sombra.
- **Seção de Benefícios** organizada em grid, com imagens lado a lado e descrições abaixo.
- **Formulário de contato** simples para captação de leads.
- **Seção de urgência** com oferta especial.
- **Rodapé** com direitos autorais.

## 🎨 Design
- Layout moderno com cores contrastantes e tipografia clara.
- Botões de chamada para ação (CTA) destacados e com animação de hover.
- Cards flutuantes com sombra suave e bordas arredondadas.
- Navegação suave entre seções (`scroll-behavior: smooth`).

## 🚀 Tecnologias Utilizadas
- **HTML5** para estrutura semântica.
- **CSS3** para estilização e responsividade.
- **GitHub Pages** para publicação online.
